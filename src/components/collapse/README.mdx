# 折叠面板

### 安装

:::card

```javascript
import { Collapse, CollapseItem } from '@byted/by-design-m';

Vue.use(Collapse).use(CollapseItem);
```

:::

## 代码演示

:::card

```topic
基础用法
```

```desc
通过`v-model`控制展开的面板列表，`activeNames`为数组格式
```

```html
<byted-collapse v-model="activeNames">
  <byted-collapse-item title="标题 1">我是内容</byted-collapse-item>
  <byted-collapse-item title="标题 2" :is-link="false" clickable>我是内容</byted-collapse-item>
  <byted-collapse-item title="标题 3" disabled>我是内容</byted-collapse-item>
</byted-collapse>
```

```javascript
export default {
  data() {
    return {
      activeNames: [],
    };
  },
};
```

:::

:::card

```topic
手风琴
```

```desc
通过`accordion`可以设置为手风琴模式，最多展开一个面板，此时`activeName`为字符串格式
```

```html
<byted-collapse v-model="activeName" accordion>
  <byted-collapse-item title="标题 1" name="1">我是内容</byted-collapse-item>
  <byted-collapse-item title="标题 2" name="2">我是内容</byted-collapse-item>
  <byted-collapse-item title="标题 3" name="3">我是内容</byted-collapse-item>
</byted-collapse>
```

```javascript
export default {
  data() {
    return {
      activeName: '1',
    };
  },
};
```

:::

:::card

```topic
高级用法
```

```html
<byted-collapse v-model="activeNames">
  <byted-collapse-item>
    <div slot="title">title slot</div>
    <p>content slot</p>
  </byted-collapse-item>
</byted-collapse>
```

```javascript
export default {
  data() {
    return {
      activeNames: [0],
    };
  },
};
```

## API

### Collapse Props

| 参数      | 说明                | 类型      | 默认值   |
| --------- | ------------------- | --------- | -------- | ------ | ------- | --- |
| v-model   | 当前展开面板的 name | `string[]` &#124; `number[]` &#124; `string` &#124; `number` | -   |
| accordion | 是否开启手风琴模式  | `boolean` | `false`  |
| border    | 是否显示外边框      | `boolean` | `true`   |

### Collapse Events

| 事件名 | 说明           | 回调参数             |
| ------ | -------------- | -------------------- | ------ |
| change | 切换面板时触发 | activeNames: `string | array` |

### CollapseItem Props

| 参数      | 说明                                 | 类型      | 默认值  |
| --------- | ------------------------------------ | --------- | ------- | ------- |
| name      | 唯一标识符，默认为索引值             | `string`   &#124; `number` | `index` |
| size      | 标题栏大小，可选值为 `large`         | `string`  | -       |
| title     | 标题栏左侧内容                       | `string` &#124; `number` | -       |
| value     | 标题栏右侧内容                       | `string` &#124; `number` | -       |
| label     | 标题栏描述信息                       | `string` &#124; `number` | -       |
| border    | 是否显示内边框                       | `boolean` | `true`  |
| clickable | 是否开启点击反馈                     | `boolean` | `false` |
| is-link   | 是否展示标题栏右侧箭头并开启点击反馈 | `boolean` | `true`  |
| disabled  | 是否禁用面板                         | `boolean` | `false` |

### CollapseItem Slots

| 名称       | 说明                          |
| ---------- | ----------------------------- |
| default    | 面板内容                      |
| value      | 自定义显示内容                |
| title      | 自定义`title`                 |
| right-icon | 自定义右侧按钮，默认是`arrow` |
