# 单元格

### 安装

:::card

```javascript
import { Cell, CellGroup } from '@byted/by-design-m';

Vue.use(Cell).use(CellGroup);
```

:::

## 代码演示

:::card

```topic
基础用法
```

```desc
`Cell`可以单独使用，也可以与`CellGroup`搭配使用。`CellGroup`可以为`Cell`提供上下外边框。
```

```html
<byted-cell-group>
  <byted-cell title="标题" value="我是内容" clickable />
  <byted-cell title="标题" value="我是内容" label="我是副标题" clickable />
  <byted-cell title="标题" value="我是内容" label="我是副标题" clickable middle />
</byted-cell-group>
```

:::

:::card

```topic
单元格大小
```

```desc
通过`size`属性可以控制单元格的大小
```

```html
<byted-cell title="标题" value="我是内容" />
<byted-cell title="标题" value="我是内容" size="large" />
```

:::

:::card

```topic
箭头方向
```

```desc
传入`is-link`属性则会在右侧显示箭头，并且可以通过传入`arrow-direction`属性控制箭头方向
```

```html
<byted-cell title="标题" is-link value="我是内容" />
<byted-cell title="标题" is-link arrow-direction="down" value="我是内容" />
```

:::

:::card

```topic
只设置 value
```

```desc
只设置`value`时会向左对齐
```

```html
<byted-cell value="我是内容" />
```

:::

:::card

```topic
分组标题
```

```desc
通过`CellGroup`的`title`属性可以指定分组标题
```

```html
<byted-cell-group title="分组1">
  <byted-cell title="标题" value="我是内容" />
</byted-cell-group>
<byted-cell-group title="分组2">
  <byted-cell title="标题" value="我是内容" size="large" />
</byted-cell-group>
```

:::

:::card

```topic
高级用法
```

```desc
如以上用法不能满足你的需求，可以使用对应的`slot`来自定义显示的内容
```

```html
<byted-cell value="我是内容" is-link>
  <template #title>
    <span>标题</span>
    <byted-icon name="radio" size="1rem" />
  </template>
</byted-cell>
<byted-cell title="标题">
  <template #right-icon>
    <byted-icon name="check-one" size="1rem" />
  </template>
</byted-cell>
```

:::

## API

### CellGroup Props

| 参数   | 说明           | 类型      | 默认值 |
| ------ | -------------- | --------- | ------ |
| title  | 分组标题       | `string`  |
| border | 是否显示外边框 | `boolean` | `true` |

### Cell Props

| 参数            | 说明                                  | 类型      | 默认值   |
| --------------- | ------------------------------------- | --------- | -------- | --- |
| title           | 左侧标题                              | `string` &#124; `number` | -   |
| value           | 右侧内容                              | `string` &#124; `number` | -   |
| label           | 标题下方的描述信息                    | `string`  | -        |
| size            | 单元格大小，可选值为 `large` `small`  | `string`  | `normal` |
| border          | 是否显示内边框                        | `boolean` | `true`   |
| clickable       | 是否开启点击反馈                      | `boolean` | `false`  |
| is-link         | 是否展示右侧箭头并开启点击反馈        | `boolean` | `false`  |
| middle          | 是否使内容垂直居中                    | `boolean` | `false`  |
| arrow-direction | 箭头方向，可选值为 `left` `up` `down` | `string`  | -        |

### Cell Events

| 事件名 | 说明             | 回调参数     |
| ------ | ---------------- | ------------ |
| click  | 点击单元格时触发 | event: Event |

### Cell Slots

| 名称       | 说明                          |
| ---------- | ----------------------------- |
| default    | 自定义`value`显示内容         |
| title      | 自定义标题显示内容            |
| label      | 自定义标题下方描述显示内容    |
| right-icon | 自定义右侧按钮，默认为`arrow` |
