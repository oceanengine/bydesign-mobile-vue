# Dropdown

### Install

:::card

```javascript
import { Dropdown, DropdownItem } from 'byted-ui';

Vue.use(Dropdown).use(DropdownItem);
```

:::

## Usage

:::card

```topic
Basic Usage
```

```html
<byted-dropdown>
  <byted-dropdown-item v-model="value1" :options="option1" />
  <byted-dropdown-item v-model="value2" :options="option2" />
</byted-dropdown>
```

```javascript
export default {
  data() {
    return {
      value1: 0,
      value2: 'a',
      option1: [
        { text: 'Option1', value: 0 },
        { text: 'Option2', value: 1 },
        { text: 'Option3', value: 2 },
      ],
      option2: [
        { text: 'Option A', value: 'a' },
        { text: 'Option B', value: 'b' },
        { text: 'Option C', value: 'c' },
      ],
    };
  },
};
```

:::card

```topic
Custom Content
```

```html
<byted-dropdown>
  <byted-dropdown-item v-model="value" :options="option" />
  <byted-dropdown-item title="title" ref="item">
    <byted-switch-cell v-model="switch1" title="title1" />
    <byted-switch-cell v-model="switch2" title="title2" />
    <byted-button size="large" type="primary" text="confirm" @click="onConfirm" />
  </byted-dropdown-item>
</byted-dropdown>
```

```javascript
export default {
  data() {
    return {
      value: 0,
      switch1: false,
      switch2: false,
      option: [
        { text: 'Option1', value: 0 },
        { text: 'Option2', value: 1 },
        { text: 'Option3', value: 2 },
      ],
    };
  },

  methods: {
    onConfirm() {
      this.$refs.item.toggle();
    },
  },
};
```

## API

### Dropdown Props

| Attribute              | Description                          | Type      | Default   |
| ---------------------- | ------------------------------------ | --------- | --------- |
| active-color           | Active color of title and option     | `string`  | `#0278FF` |
| inactive-color         | Inactive color of title and option   | `string`  | `#333`    |
| z-index                | z-index of menu item                 | `number`  | `10`      |
| duration               | Transition duration, unit second     | `number`  | `0.2`     |
| direction              | Expand direction, can be set to `up` | `string`  | `down`    |
| overlay                | Whether to show overlay              | `boolean` | `true`    |
| close-on-click-overlay | Whether to close when click overlay  | `boolean` | `true`    |
| close-on-click-outside | Whether to close when click outside  | `boolean` | `true`    |

### DropdownItem Props

| Attribute | Description                                | Type       | Default                 |
| --------- | ------------------------------------------ | ---------- | ----------------------- | --- |
| value     | Value of current optionï¼Œcan use `v-model` | `string` &#124; `number` | -   |
| title     | Item title                                 | `string`   | Text of selected option |
| options   | Options                                    | `Option[]` | `[]`                    |
| disabled  | Whether to disable dropdown item           | `boolean`  | `false`                 |

### DropdownItem Events

| Event  | Description                               | Arguments |
| ------ | ----------------------------------------- | --------- |
| change | Triggered select option and value changed | value     |
| open   | Triggered when open menu                  | -         |
| opened | Triggered when opened menu                | -         |
| close  | Triggered when close menu                 | -         |

### DropdownItem Slots

| Name    | Description                  |
| ------- | ---------------------------- |
| default | default DropdownItem content |
| title   | Custom title content         |

### Methods

Use ref to get DropdownItem instance and call instance methods

| Name   | Attribute     | Return value | Description    |
| ------ | ------------- | ------------ | -------------- |
| toggle | show: boolean | -            | Toggle display |

### Option

| Key   | Description | Type     |
| ----- | ----------- | -------- | ------- |
| text  | Text        | `string` |
| value | Value       | `string` &#124; `number` |
