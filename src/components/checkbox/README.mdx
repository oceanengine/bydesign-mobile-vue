# 复选框

### 安装

:::card

```javascript
import { Checkbox, CheckboxGroup } from '@byted/by-design-m';

Vue.use(Checkbox).use(CheckboxGroup);
```

:::

## 代码演示

:::card

```topic
基础用法
```

```desc
通过`v-model`绑定 checkbox 的勾选状态
```

```html
<byted-checkbox-group v-model="checked">
  <byted-checkbox v-for="index in 4" :key="index" :value="index">
    复选框{{ index }}
  </byted-checkbox>
</byted-checkbox-group>
```

```javascript
export default {
  data() {
    return {
      checked: [1, 3],
    };
  },
};
```

## API

### Checkbox Props

| 参数            | 说明                        | 类型      | 默认值  |
| --------------- | --------------------------- | --------- | ------- | -------- | ------- | --- |
| v-model         | 是否为选中状态              | `boolean` | `false` |
| disabled        | 是否禁用复选框              | `boolean` | `false` |
| label-position  | 文本位置，可选值为 `left`   | `string`  | `right` |
| size            | 设置型号大小，默认为 normal | `string`  | `small` | `normal` | `large` | -   |
| stopPropagation | 阻止事件冒泡， 默认为 false | `boolean` | `false` | -        |

### CheckboxGroup Props

| 参数     | 说明                                         | 类型      | 默认值    |
| -------- | -------------------------------------------- | --------- | --------- | -------- | ------- | --- |
| v-model  | 所有选中项的标识符                           | `any[]`   | -         |
| disabled | 是否禁用所有复选框                           | `boolean` | `false`   |
| max      | 设置最大可选数，0 为无限制                   | `number`  | `0`       |
| size     | 设置型号大小，默认为 normal                  | `string`  | `small`   | `normal` | `large` | -   |
| type     | 类型，可选值为 `default`、`number`、`square` | `string`  | `default` |

## Events

### Checkbox Events

| 事件名 | 说明                     | 回调参数     |
| ------ | ------------------------ | ------------ |
| change | 当绑定值变化时触发的事件 | 当前组件的值 |

### CheckboxGroup Events

| 事件名 | 说明                     | 回调参数     |
| ------ | ------------------------ | ------------ |
| change | 当绑定值变化时触发的事件 | 当前组件的值 |

# 复选框组

:::card

```topic
基础用法
```

```desc
通过`v-model`绑定值当前选中项的value, 默认为多选。value值必须为Array类型
```

```html
<byted-checker v-model="checker">
  <byted-row gutter="10">
    <byted-col span="8">
      <byted-checker-item label="0" :value="0" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item label="1" value="1" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item value="2">2</byted-checker-item>
    </byted-col>
  </byted-row>
</byted-checker>

<byted-checker v-model="checker" round>
  <byted-row gutter="10">
    <byted-col span="8">
      <byted-checker-item label="0" :value="0" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item label="1" value="1" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item value="2">2</byted-checker-item>
    </byted-col>
  </byted-row>
</byted-checker>
```

```javascript
data() {
  return {
    checker: [0, '2']
  }
}
```

:::

## API

### Checker Props

| 参数      | 说明                          | 类型                      | 默认值     |
| --------- | ----------------------------- | ------------------------- | ---------- |
| inputType | 可选值为：`radio` `checkbox`  | `string`                  | `checkbox` |
| value     | 类型 Array String 或者 Number | `array` `string` `number` | `[]`       |
| type      | 可选值为：`primary`           | `string`                  | `default`  |
| size      | 可选值为：`small` `normal`    | `string`                  | `normal`   |

### CheckerItem Props

| 参数  | 说明                          | 类型                      | 默认值 |
| ----- | ----------------------------- | ------------------------- | ------ |
| value | 类型 Array String 或者 Number | `array` `string` `number` | `[]`   |
| label | 内容                          | `string`                  | `''`   |
| name  | 标示                          | `string`                  | `''`   |
| info  | 显示信息                      | `string`                  | `''`   |

### Events

| 事件名     | 说明                     | 回调参数           |
| ---------- | ------------------------ | ------------------ |
| change     | 当绑定值变化时触发的事件 | 当前选中项的 value |
| input      | 单选异步事件             | 要选中项的 value   |
| click      | 点击时触发               | event              |
| touchstart | 点击时触发               | event              |
