# Checkbox

### Install

:::card

```javascript
import { Checkbox, CheckboxGroup } from '@byted/by-design-m';

Vue.use(Checkbox).use(CheckboxGroup);
```

:::

## Usage

:::card

```topic
Basic Usage
```

```html
<byted-checkbox-group v-model="checked">
  <byted-checkbox v-for="index in 4" :key="index" :value="index">
    checkbox{{ index }}
  </byted-checkbox>
</byted-checkbox-group>
```

```javascript
export default {
  data() {
    return {
      checked: [1, 3],
    };
  },
};
```

## API

### Checkbox Props

| Attribute       | Description          | Type      | Default |
| --------------- | -------------------- | --------- | ------- | -------- | ------- | --- |
| v-model         | Check status         | `boolean` | `false` | -        |
| disabled        | Diable checkbox      | `boolean` | `false` | -        |
| label-position  | Can be set to `left` | `string`  | `right` | -        |
| size            | Size status          | `string`  | `small` | `normal` | `large` | -   |
| stopPropagation | stopPropagation      | `boolean` | `false` | -        |

### CheckboxGroup Props

| Attribute | Description                       | Type      | Default   |
| --------- | --------------------------------- | --------- | --------- | -------- | ------- | --- |
| v-model   | Names of all checked checkboxes   | `any[]`   | -         | -        |
| disabled  | Disable all checkboxes            | `boolean` | `false`   | -        |
| max       | Maximum amount of checked options | `number`  | `0`       | -        |
| size      | Size status                       | `string`  | `small`   | `normal` | `large` | -   |
| type      | Type                              | `string`  | `default` | `number` |

## Events

### Checkbox Events

| Event  | Description                  | Arguments     |
| ------ | ---------------------------- | ------------- |
| change | Triggered when value changed | current value |

### CheckboxGroup Events

| Event  | Description                  | Arguments     |
| ------ | ---------------------------- | ------------- |
| change | Triggered when value changed | current value |

# 复选框组

:::card

```topic
基础用法
```

```desc
通过`v-model`绑定值当前选中项的value, 默认为多选。value值必须为Array类型
```

```html
<byted-checker v-model="checker">
  <byted-row gutter="10">
    <byted-col span="8">
      <byted-checker-item label="0" :value="0" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item label="1" value="1" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item value="2">2</byted-checker-item>
    </byted-col>
  </byted-row>
</byted-checker>

<byted-checker v-model="checker" round>
  <byted-row gutter="10">
    <byted-col span="8">
      <byted-checker-item label="0" :value="0" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item label="1" value="1" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item value="2">2</byted-checker-item>
    </byted-col>
  </byted-row>
</byted-checker>
```

```javascript
data() {
  return {
    checker: [0, '2']
  }
}
```

## API

### Checker Props

| 参数      | 说明                          | 类型                      | 默认值     |
| --------- | ----------------------------- | ------------------------- | ---------- |
| inputType | 可选值为：`radio` `checkbox`  | `string`                  | `checkbox` |
| value     | 类型 Array String 或者 Number | `array` `string` `number` | `[]`       |
| type      | 可选值为：`primary`           | `string`                  | `default`  |
| size      | 可选值为：`small` `normal`    | `string`                  | `normal`   |

### CheckerItem Props

| 参数  | 说明                          | 类型                      | 默认值 |
| ----- | ----------------------------- | ------------------------- | ------ |
| value | 类型 Array String 或者 Number | `array` `string` `number` | `[]`   |
| label | 内容                          | `string`                  | `''`   |
| name  | 标示                          | `string`                  | `''`   |
| info  | 显示信息                      | `string`                  | `''`   |

### Events

| 事件名     | 说明                     | 回调参数           |
| ---------- | ------------------------ | ------------------ |
| change     | 当绑定值变化时触发的事件 | 当前选中项的 value |
| input      | 单选异步事件             | 要选中项的 value   |
| click      | 点击时触发               | event              |
| touchstart | 点击时触发               | event              |
