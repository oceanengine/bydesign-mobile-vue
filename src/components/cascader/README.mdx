# 多级级联单选

### 安装

:::card

```javascript
import { cascader } from '@byted/by-design-m';

Vue.use(cascader);
```

:::

## 代码演示

:::card

```topic
基本类型
```

```html
<byted-cascader :options="options" v-model="selected" @change="handleChange" />
```

:::

:::card

```topic
加collapse（展开三列及以上时收起第一列）
```

```html
<byted-cascader :options="options" v-model="selected" collapse @change="handleChange" />
```

:::

:::card

```topic
异步加载子节点
```

```html
<byted-cascader
  :options="options2"
  v-model="selected"
  collapse
  @active-item-change="handleActiveItemChange"
  @change="handleChange"
/>
```

:::

## API

### Props

| Attribute    | Description                                 | Type      | Default |
| ------------ | ------------------------------------------- | --------- | ------- |
| options      | 选项数据源, 键名包括 label、value、children | `array`   | -       |
| value        | 选中项绑定值                                | `array`   | []      |
| collapse     | 大于两列时，是否需要收缩第一列宽度          | `boolean` | `false` |
| selectedItem | 默认选中项的 value 数组                     | `array`   | []      |

### Events

| Event              | 说明                       | 参数类型                  | 参数说明                                                      |
| ------------------ | -------------------------- | ------------------------- | ------------------------------------------------------------- |
| change             | 根节点和父节点点击均触发   | array（selectlabelArray） | 若根节点点击：选中项 label 属性的全路径；若父节点点击：返回[] |
| input              | 根节点和父节点点击均触发   | array（selectArray）      | 若根节点点击：选中项所有属性的全路径；若父节点点击：返回[]    |
| active-item-change | 当父级选项变化时触发的事件 | array（selectValueArray） | 各父级选项组成的数组                                          |

### selectArray

:::card

```javascript
this.$emit('input', selectArray);
// selectArray = [{label: '动物', value: '123' },
//   {label: '狗', value: '456'},
//   {label: '金毛', value: '789'}]
```

:::

### selectlabelArray

:::card

```javascript
this.$emit('change', selectlabelArray);
// selectlabelArray = ['动物', '狗'，'金毛']
```
