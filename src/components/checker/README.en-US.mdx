# checker

### Install

:::card

```javascript
import { Checker, CheckerItem } from '@byted/by-design-m';

Vue.use(Checker).use(CheckerItem);
```

:::

## Usage

:::card

```topic
Basic Usage
```

```desc
inputType defalutValue is 'checkbox'. value must be Array
```

```html
<byted-checker v-model="checker">
  <byted-row gutter="10">
    <byted-col span="8">
      <byted-checker-item label="0" :value="0" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item label="1" value="1" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item label="2" value="2" />
    </byted-col>
  </byted-row>
</byted-checker>
```

```javascript
data() {
  return {
    checker: [0, '2']
  }
}
```

```topic
Radio Async
```

```desc
inputType is radio，@input + value
```

```html
<byted-checker inputType="radio" :value="checkerAsync" @input="onInput">
  <byted-row gutter="10">
    <byted-col span="8">
      <byted-checker-item label="0" :value="0" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item label="1" value="1" />
    </byted-col>
    <byted-col span="8">
      <byted-checker-item label="2" value="2" />
    </byted-col>
  </byted-row>
</byted-checker>
```

```javascript
data() {
  return {
    checkerAsync: 0,
  }
},
methods: {
  onInput(value) {
    this.$confirm(`Do you want to chose ${value}？`).then(() => {
      this.checkerAsync = value;
    }).catch(() => {
      //
    });
  }
}
```

## API

### Checker Props

| Attribute | Description                             | Type     | Default    |
| --------- | --------------------------------------- | -------- | ---------- |
| inputType | Can be set to `radio`                   | `string` | `checkbox` |
| value     | Can be set to `Array` `String` `Number` | `array`  | `[]`       |
| type      | Can be set to `primary`                 | `string` | `default`  |
| size      | Can be set to `small` `normal`          | `string` | `normal`   |

### CheckerItem Props

| Attribute | Description                 | Type                      | Default |
| --------- | --------------------------- | ------------------------- | ------- |
| value     | Type Array String or Number | `array` `string` `number` | `[]`    |
| label     | label as content            | `string`                  | `''`    |
| name      | name                        | `string`                  | `''`    |
| info      | show info                   | `string`                  | `''`    |

### Events

| Event      | Description        | Arguments |
| ---------- | ------------------ | --------- |
| change     | when value changed | value     |
| input      | when you use async | value     |
| click      | click              | event     |
| touchstart | touchstart         | event     |
