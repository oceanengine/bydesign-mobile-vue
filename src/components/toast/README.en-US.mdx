# Toast

### Usage

:::card

```topic
Text
```

```javascript
vue.$toast('Some messages');
```

:::

:::card

```topic
Success/Fail/Warning/Loading
```

```javascript
vue.$toast({
  message: 'success',
  type: 'success',
});
```

:::

:::card

```topic
Set Time
```

```javascript
Vue.$toast({
  message: 'ByDesign',
  duration: 1000,
});
```

:::

:::card

```topic
ForbidClick
```

```javascript
Vue.$toast({
  message: 'forbidClick',
  forbidClick: true,
});
```

:::

:::card

```topic
set private function name
```

```javascript
import { Toast } from '@byted/by-design-m';

Vue.use(Toast, {
  funcName: 'tell',
});

Vue.$tell('some message');
```

:::

## API

### Options

| Attribute   | Description                                       | Type       | Default |
| ----------- | ------------------------------------------------- | ---------- | ------- |
| type        | Can be set to `success` `error` `warning`         | `string`   | `text`  |
| message     | Message                                           | `string`   | `''`    |
| forbidClick | Whether to forbid click background                | `boolean`  | `false` |
| duration    | Toast duration(ms), won't disappear if value is 0 | `number`   | `3000`  |
| onOpened    | Callback function after opened                    | `Function` | -       |
| onClose     | Callback function after close                     | `Function` | -       |
