# Radio

### Install

:::card

```javascript
import { Radio, RadioGroup } from '@byted/by-design-m';

Vue.use(Radio).use(RadioGroup);
```

:::

## Usage

:::card

```topic
Basic Usage
```

```html

  <byted-radio value="1">radio 1</byted-radio>

  <byted-radio value="3">radio 3</byted-radio>
</byted-radio-group>
```

```javascript
export default {
```

```topic
Async Usage
```

```html
<byted-radio-group :value="radio" @input="onInput" @click="onClick" @change="onChange">
  <byted-radio value="1">radio 1</byted-radio>
  <byted-radio value="2">radio 2</byted-radio>
  <byted-radio value="3">radio 3</byted-radio>
</byted-radio-group>
```

```javascript
export default {
  data() {
    return {
      radio: '1',
    };
  },
  methods: {
    onInput(value) {
      console.log('input');
      this.$confirm(`Do you wang to chose ${value}？`)
        .then(() => {
          this.radio = value;
        })
        .catch(() => {
          //
        });
    },
    onClick() {
      console.log('click');
    },
    onChange(value) {
      console.log('change', value);
    },
  },
};
```

## API

### Radio Props

| Attribute       | Description                    | Type      | Default  |
| --------------- | ------------------------------ | --------- | -------- | --------- | ------- |
| disabled        | Disabled                       | `boolean` | `false`  |
| label-position  | Can be set to `left`           | `string`  | `right`  |
| size            | Size                           | `string`  | `small`  | `normal`  | `large` |
| type            | Type: `default`、`dot`、`hook` |           | `string` | `default` |
| stopPropagation | stopPropagation                | `boolean` | `false`  |

### RadioGroup Props

| Attribute | Description                              | Type                 | Default  |
| --------- | ---------------------------------------- | -------------------- | -------- | --------- | ------- |
| value     | Name of checked radio                    | `string` or `number` | -        |
| disabled  | Diable all radios                        | `boolean`            | `false`  |
| size      | Size                                     | `string`             | `small`  | `normal`  | `large` |
| type      | Type: `default`、`dot`、`hook`、`circle` |                      | `string` | `default` |

## Events

### RadioGroup Events

| Event      | Description                  | Parameters    |
| ---------- | ---------------------------- | ------------- |
| change     | Triggered when value changed | current value |
| input      | when you use async           | value         |
| click      | click                        | event         |
| touchstart | touchstart                   | event         |

## API

### Checker Props

| Attribute | Description                             | Type     | Default    |
| --------- | --------------------------------------- | -------- | ---------- |
| inputType | Can be set to `radio`                   | `string` | `checkbox` |
| value     | Can be set to `Array` `String` `Number` | `array`  | `[]`       |
| type      | Can be set to `primary`                 | `string` | `default`  |
| size      | Can be set to `small` `normal`          | `string` | `normal`   |

### CheckerItem Props

| Attribute | Description                 | Type                      | Default |
| --------- | --------------------------- | ------------------------- | ------- |
| value     | Type Array String or Number | `array` `string` `number` | `[]`    |
| label     | label as content            | `string`                  | `''`    |
| name      | name                        | `string`                  | `''`    |
| info      | show info                   | `string`                  | `''`    |

### Events

| Event      | Description        | Arguments |
| ---------- | ------------------ | --------- |
| change     | when value changed | value     |
| input      | when you use async | value     |
| click      | click              | event     |
| touchstart | touchstart         | event     |
