# 单选框

### 安装

:::card

```javascript
import { Radio, RadioGroup } from '@byted/by-design-m';

Vue.use(Radio).use(RadioGroup);
```

:::

## 代码演示

:::card

```topic
基础用法
```

```desc
通过`v-model`绑定值当前选中项的 value
```

```html
<byted-radio-group v-model="radio">
  <byted-radio value="1">单选框 1</byted-radio>
  <byted-radio value="2">单选框 2</byted-radio>
  <byted-radio value="3">单选框 3</byted-radio>
</byted-radio-group>
```

```javascript
export default {
  data() {
    return {
      radio: '1',
    };
  },
};
```

```topic
异步用法
```

```html
<byted-radio-group :value="radio" @input="onInput" @click="onClick" @change="onChange">
  <byted-radio value="1">单选框 1</byted-radio>
  <byted-radio value="2">单选框 2</byted-radio>
  <byted-radio value="3">单选框 3</byted-radio>
</byted-radio-group>
```

```javascript
export default {
  data() {
    return {
      radio: '1',
    };
  },
  methods: {
    onInput(value) {
      console.log('input');
      this.$confirm(`是否确定选择${value}选项？`)
        .then(() => {
          this.radio = value;
        })
        .catch(() => {
          //
        });
    },
    onClick() {
      console.log('click');
    },
    onChange(value) {
      console.log('change', value);
    },
  },
};
```

## API

### Radio Props

| 参数            | 说明                                    | 类型      | 默认值    |
| --------------- | --------------------------------------- | --------- | --------- | -------- | ------- |
| disabled        | 是否为禁用状态                          | `boolean` | `false`   |
| label-position  | 文本位置，可选值为 `left`               | `string`  | `right`   |
| size            | 设置型号大小，默认为`normal`            | `string`  | `small`   | `normal` | `large` |
| type            | 类型，可选值为 `default`、`dot`、`hook` | `string`  | `default` |
| stopPropagation | 阻止事件冒泡， 默认为`false`            | `boolean` | `false`   |

### RadioGroup Props

| 参数     | 说明                                              | 类型                 | 默认值    |
| -------- | ------------------------------------------------- | -------------------- | --------- | -------- | ------- |
| value    | 当前选中项的值                                    | `string` or `number` | -         |
| disabled | 是否禁用所有单选框                                | `boolean`            | `false`   |
| size     | 设置型号大小，默认为`normal`                      | `string`             | `small`   | `normal` | `large` |
| type     | 类型，可选值为 `default`、`dot`、`hook`、`circle` | `string`             | `default` |

## Events

### RadioGroup Events

| 事件名     | 说明                     | 回调参数           |
| ---------- | ------------------------ | ------------------ |
| change     | 当绑定值变化时触发的事件 | 当前选中项的 value |
| input      | 单选异步事件             | 要选中项的 value   |
| click      | 点击时触发               | event              |
| touchstart | 点击时触发               | event              |

## API

### Checker Props

| 参数      | 说明                          | 类型                      | 默认值     |
| --------- | ----------------------------- | ------------------------- | ---------- |
| inputType | 可选值为：`radio` `checkbox`  | `string`                  | `checkbox` |
| value     | 类型 Array String 或者 Number | `array` `string` `number` | `[]`       |
| type      | 可选值为：`primary`           | `string`                  | `default`  |
| size      | 可选值为：`small` `normal`    | `string`                  | `normal`   |

### CheckerItem Props

| 参数  | 说明                          | 类型                      | 默认值 |
| ----- | ----------------------------- | ------------------------- | ------ |
| value | 类型 Array String 或者 Number | `array` `string` `number` | `[]`   |
| label | 内容                          | `string`                  | `''`   |
| name  | 标示                          | `string`                  | `''`   |
| info  | 显示信息                      | `string`                  | `''`   |

### Events

| 事件名     | 说明                     | 回调参数           |
| ---------- | ------------------------ | ------------------ |
| change     | 当绑定值变化时触发的事件 | 当前选中项的 value |
| input      | 单选异步事件             | 要选中项的 value   |
| click      | 点击时触发               | event              |
| touchstart | 点击时触发               | event              |
