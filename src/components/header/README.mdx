# 自定义头部

### 安装

:::card

```javascript
import { Header } from '@byted/by-design-m';

Vue.use(Header);
```

:::

## 代码演示

:::card

```topic
基本用法
```

```html
<byted-header title="Title" @click-left="onClickLeft" />
```

```javascript
export default {
  methods: {
    onClickLeft() {
      Toast('Back');
    },
  },
};
```

:::

:::card

```topic
顶部固定
```

```html
<byted-header title="Title" fixed @click-left="back" />
```

:::

:::card

```topic
无返回按钮
```

```html
<byted-header title="ByDesign" :left-arrow="false" />
```

:::

:::card

```topic
自定义按钮
```

```html
<byted-header title="ByDesign">
  <byted-button slot="left" text="取消" type="text" />
  <byted-button slot="right" text="确定" type="text" />
</byted-header>
```

:::

:::card

:::card

```topic
高级用法
```

```html
<byted-header title="Title">
```

## API

### Props

| Attribute  | Description                                      | Type      | Default   |
| ---------- | ------------------------------------------------ | --------- | --------- |
| type       | Can be set to `primary`                          | `string`  | `default` |
| title      | Title                                            | `string`  | `''`      |
| left-arrow | Whether to show left arrow                       | `boolean` | `false`   |
| border     | when `type` === `default` Whether to show border | `boolean` | `true`    |
| fixed      | Whether to fixed top                             | `boolean` | `false`   |

### Slots

| Name  | Description               |
| ----- | ------------------------- |
| title | Custom title              |
| left  | Custom left side content  |
| right | Custom right side content |

### Events

| Event       | Description                       | Arguments |
| ----------- | --------------------------------- | --------- |
| click-left  | Triggered when click left button  | -         |
| click-right | Triggered when click right button | -         |
