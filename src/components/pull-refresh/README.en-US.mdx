# PullRefresh

### Install

:::card

```javascript
import { PullRefresh } from '@byted/by-design-m';

Vue.use(PullRefresh);
```

:::

## Usage

:::card

```topic
Basic Usage
```

```html
<byted-pull-refresh
  v-model="isLoading"
  :load-end-status="loadEndStatus"
  type="primary"
  size="large"
  foot-disabled
  style="background: #fff"
  @pull-down="pullDown"
>
  <div class="content" style="justify-content: flex-start">
    <!-- <span>{{ t('refreshStatus') }}： {{ isLoading }}</span> -->
    <span>{{ t('pullDownStatus') }}</span>
  </div>
</byted-pull-refresh>

<byted-pull-refresh
  v-model="isPullUpLoading"
  :load-end-status="loadEndStatus"
  headDisabled
  type="primary"
  size="large"
  style="background: #fff"
  @pull-up="pullUp"
>
  <div class="content" style="justify-content: flex-end">
    <!-- <span>{{ t('refreshStatus') }}： {{ isLoading }}</span> -->
    <span>{{ t('pullUpStatus') }}</span>
  </div>
</byted-pull-refresh>
```

```javascript
export default {
  data() {
    return {
      isLoading: false,
      isPullUpLoading: false,
      loadEndStatus: '',
      titleStyle: {
        'padding-left': '0.8rem',
      },
    };
  },

  methods: {
    pullDown() {
      setTimeout(() => {
        this.isLoading = false;
        this.loadEndStatus = 'success';
      }, 1100);
    },
    pullUp() {
      setTimeout(() => {
        this.isPullUpLoading = false;
        this.loadEndStatus = 'success';
      }, 1100);
    },
  },
};
```

## API

### Props

| Attribute              | Description                                                                            | Type      | Default            |
| ---------------------- | -------------------------------------------------------------------------------------- | --------- | ------------------ |
| v-model                | Loading status                                                                         | `boolean` | -                  |
| type                   | type `default`、`primary`、`circle`                                                    | `string`  | `default`          |
| size                   | size `normal`、`large`                                                                 | `string`  | `normal`           |
| pulling-text           | Text to show when pulling                                                              | `string`  | `下拉刷新`         |
| loosing-text           | Text to show when loosing                                                              | `string`  | `释放刷新`         |
| loading-text           | Text to show when loading                                                              | `string`  | `加载中...`        |
| success-text           | Text to show when success                                                              | `string`  | `刷新成功`         |
| fail-text              | Text to show when fail                                                                 | `string`  | `刷新失败`         |
| foot-loading-text      | Text to show when pull up                                                              | `string`  | `正在加载`         |
| foot-load-success-text | Text to show when pull up success                                                      | `string`  | `加载成功`         |
| foot-load-fail-text    | Text to show when pull up fail                                                         | `string`  | `加载失败, 请重试` |
| foot-load-end-text     | Text to show when pull up end                                                          | `string`  | `没有更多啦`       |
| load-end-status        | push refresh end status `success`、`fail`                                              | `string`  | -                  |
| foot-load-status       | pull down load status `footLoadSuccess`、`footLoadFail` 、`footLoadEnd`、`footLoading` | `string`  | -                  |
| end-duration           | End time duration                                                                      | `number`  | `500`              |
| animation-duration     | Animation duration                                                                     | `number`  | `300`              |
| head-height            | Height of head                                                                         | `number`  | `50`               |
| head-top               | Position absoult top of text to show when pulling                                      | `number`  | `-44`              |
| pull-up-height         | Trigger height before pull up loading                                                  | `number`  | `35`               |
| disabled               | Disabled pull refresh                                                                  | `boolean` | `false`            |
| footDisabled           | Disabled pull down                                                                     | `boolean` | `false`            |

### Events

| Event     | Description                 | Arguments |
| --------- | --------------------------- | --------- |
| pull-down | Triggered when pull refresh | -         |
| pull-up   | Triggered when pull Up      | -         |

### Slots

| Name            | Description                                   | scoped-slot  |
| --------------- | --------------------------------------------- | ------------ |
| default         | Default slot                                  | -            |
| icon            | 自定义刷新图标                                |
| normal          | Content of head when at normal status         | -            |
| pulling         | Content of head when at pulling               | { distance } |
| loosing         | Content of head when at loosing               | { distance } |
| loading         | Content of head when at loading               | { distance } |
| success         | Content of head when at success               | { distance } |
| fail            | Content of head when at fail                  | { distance } |
| ------          | ------                                        | ------       |
| footNormal      | Content of head when at pull up normal status | -            |
| footLoading     | Content of head when at pull up loading       | -            |
| footLoadSuccess | Content of head when at pull up success       | -            |
| footLoadFail    | Content of head when at pull up fail          | -            |
| footLoadEnd     | Content of head when at pull down load end    | -            |
