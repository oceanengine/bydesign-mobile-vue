# Search

### Install

:::card

```javascript
import { Search } from '@byted/by-design-m';

Vue.use(Search);
```

:::

## Usage

## API

### Props

| Attribute    | Description                       | Type              | Default      |
| ------------ | --------------------------------- | ----------------- | ------------ | --- |
| value        | Can be set to `String` `Number`   | `string` &#124; `number` | -   |
| clearable    | Whether to enable clear control   | `Boolean`         | `true`       |
| cancelable   | Whether to enable cancel control  | `Boolean`         | `false`      |
| borderRadius | border-radius                     | `Number` `String` | `0`          |
| placeholder  | Tips for filling out input fields | `String`          | `请输入内容` |
| autofocus    | autofocus                         | `boolean`         | `false`      |

### Events

| Event  | Description                                             | Arguments            |
| ------ | ------------------------------------------------------- | -------------------- |
| input  | Triggered when input value changed                      | value: current value |
| search | Triggered when the keyboard confirmation key is clicked | -                    |
| cancel | Triggered when click cancel text                        | -                    |
| focus  | Triggered when input gets focus                         | event: Event         |
| blur   | Triggered when input loses focus                        | event: Event         |
| clear  | Triggered when click clear icon                         | event: Event         |
| click  | Triggered when click Field                              | event: Event         |

### Slots

| Name  | Attribute    |
| ----- | ------------ |
| left  | Custom left  |
| icon  | Custom icon  |
| right | Custom right |
