# Upload

### Install

:::card

```javascript
import { upload } from '@byted/by-design-m';

Vue.use(upload);
```

:::

## Usage

:::card

```topic
Default
```

```html
<byted-upload
  :action="URL"
  :before-upload="beforeUpload"
  :on-success="onSuccess"
  :on-error="onError"
  accept="image/png,image/jpeg,image/jpg"
/>
```

:::

## API

### Props

| Attribute         | Description                                   | Type      | Default    |
| ----------------- | --------------------------------------------- | --------- | ---------- |
| action            | upload link                                   | `string`  | ''         |
| name              | Input name                                    | `string`  | 'Filedata' |
| data              | post data                                     | `object`  | {}         |
| headers           | http header                                   | `object`  | {}         |
| accept            | accept file type                              | `string`  | ''         |
| disabled          | Whether to disabled the upload                | `Boolean` | false      |
| multiple          | Whether to enable multiple selection pictures | `Boolean` | false      |
| maxSize           | Max size of file，unit is byte                | `Number`  | false      |
| exceedMaxSizeText | exceed max size toast text                    | `String`  | false      |
| maxCount          | Max count of image                            | `Number`  | 10         |
| withCredentials   | Whether to with cookie on CORS http request   | `Boolean` | false      |
| capture           | 同input元素的capture           | `String` | null      |

### Events

| Event        | Description                        | Arguments                               |
| ------------ | ---------------------------------- | --------------------------------------- |
| beforeUpload | Triggered when file before upload  | file: file Object                       |
| onSuccess    | Triggered when file upload success | res: response object，file: file Object |
| onError      | Triggered when file upload fail    | err: error object，file: file Object    |
