# 进度条

### 安装

:::card

```javascript
import { Progress } from '@byted/by-design-m';

Vue.use(Progress);
```

:::

## 代码演示

:::card

```topic
基础用法
```

```html
<byted-progress :percentage="60" />
<byted-progress :percentage="100" color="#6ABF40" />
<byted-progress :percentage="70" :inactive="false" />
```

## API

### Props

| 参数         | 描述                                           | 类型      | 默认值    |
| ------------ | ---------------------------------------------- | --------- | --------- |
| percentage   | progress show percentage                       | `number`  | `50`      |
| trackColor   | progress background color                      | `string`  | `#EBEBEB` |
| color        | progress percentage show part background color | `string`  | `#338AFF` |
| inactive     | is in active                                   | `boolean` | `true`    |
| textInMiddle | is show percentage text in progress middle     | `boolean` | `false`   |
