# 滑块

### 安装

:::card

```javascript
import { Slider } from '@byted/by-design-m';

Vue.use(Slider);
```

:::

## 代码演示

:::card

```topic
基础用法
```

```desc
通过 `v-model` 绑定滑块的值
```

```html
<byted-slider v-model="value" @change="onChange" />
```

```javascript
data() {
  return {
    value: 50
  }
},

methods: {
  onChange () {
    this.$toast(this.value)
  }
}
```

:::

:::card

```topic
指定范围
```

```desc
使用 `min` `max` 指定滑块的最大值和最小值
```

```html
<byted-slider v-model="value" :min="-50" :max="50" @change="onChange" />
```

```javascript
data() {
  return {
    value: 50
  }
},

methods: {
  onChange () {
    this.$toast(this.value)
  }
}
```

:::

:::card

```topic
指定步长
```

```desc
使用 `step` 指定滑块每次滑动的最小值
```

```html
<byted-slider v-model="value" :step="10" @change="onChange" />
```

```javascript
data() {
  return {
    value: 50
  }
}
```

:::

:::card

```topic
指定步长值
```

```desc
使用 `steps` 为数字数组，要求 steps.length >= 2, 滑块会被等分成（steps.length - 1）份
```

```html
<byted-slider v-model="value" :steps="steps" @change="onChange" />
```

```javascript
data() {
  return {
    value: 70,
    steps: [0, 50, 70, 77, 90, 120]
  }
}
```

:::

:::card

```topic
禁用态
```

```html
<byted-slider v-model="value" disabled />
```

```javascript
data() {
  return {
    value: 50
  }
}
```

:::

## API

### Props

| 参数           | 说明             | 类型      | 默认值    |
| -------------- | ---------------- | --------- | --------- | --- |
| value          | 当前输入的值     | `string` &#124; `number` |
| max            | 最大值           | `Number`  | `100`     |
| min            | 最小值           | `Number`  | `0`       |
| step           | 步长             | `Number`  | `1`       |
| active-color   | 进度条激活态颜色 | `String`  | `#0278FF` |
| inactive-color | 进度条默认颜色   | `String`  | `#dcdcdc` |
| disabled       | 是否禁用滑块     | `boolean` | `false`   | -   |

### Events

| 事件       | 说明             | 回调参数        |
| ---------- | ---------------- | --------------- |
| change     | 进度值改变后触发 | value: 当前进度 |
| drag-start | 开始拖动时触发   | -               |
| drag-end   | 结束拖动时触发   | -               |
