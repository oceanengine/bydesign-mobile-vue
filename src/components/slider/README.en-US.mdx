# Slider

### Install

:::card

```javascript
import { Slider } from '@byted/by-design-m';

Vue.use(Slider);
```

:::

## Usage

:::card

```topic
Basic Usage
```

```html
<byted-slider v-model="value" @change="onChange" />
```

```javascript
data() {
  return {
    value: 50
  }
},

methods: {
  onChange () {
    this.$toast(this.value)
  }
}
```

:::

:::card

```topic
Range Usage
```

```html
<byted-slider v-model="value" :min="-50" :max="50" @change="onChange" />
```

```javascript
data() {
  return {
    value: 50
  }
},

methods: {
  onChange () {
    this.$toast(this.value)
  }
}
```

:::

:::card

```topic
Step Usage
```

```html
<byted-slider v-model="value" :step="10" @change="onChange" />
```

```javascript
data() {
  return {
    value: 50
  }
}
```

:::

:::card

```topic
指定步长值
```

```desc
use `steps` as Array, steps.length >= 2, slider was spliced to（steps.length - 1）
```

```html
<byted-slider v-model="value" :steps="steps" @change="onChange" />
```

```javascript
data() {
  return {
    value: 70,
    steps: [0, 50, 70, 77, 90, 120]
  }
}
```

:::

## API

### Props

| Attribute      | Description               | Type      | Default   |
| -------------- | ------------------------- | --------- | --------- |
| value          | Current value             | `string` &#124; `number` |
| max            | Max value                 | `Number`  | `100`     |
| min            | Min value                 | `Number`  | `0`       |
| step           | Step size                 | `Number`  | `1`       |
| active-color   | Active color of bar       | `String`  | `#0278FF` |
| inactive-color | Inactive color of bar     | `String`  | `#dcdcdc` |
| disabled       | Whether to disable slider | `boolean` | `false`   |

### Events

| Event      | Description                  | Arguments           |
| ---------- | ---------------------------- | ------------------- |
| change     | Triggered after value change | value: current rate |
| drag-start | Triggered when start drag    | -                   |
| drag-end   | Triggered when end drag      | -                   |
