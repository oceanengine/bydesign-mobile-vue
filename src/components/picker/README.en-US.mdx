# Picker

### Install

:::card

```javascript
import { Picker } from '@byted/by-design-m';

Vue.use(Picker);
```

:::

## Usage

:::card

```topic
Basic Usage
```

```desc
single `value` is `String` or `Number`
single item of `options` is `Object`, key should be `label`、`value`
```

```html
<byted-picker title="title" :value="value" :options="options" />
```

```javascript
export default {
  data() {
    return {
      value: 2,
      options: [
        {
          label: 0,
          value: 0,
        },
        {
          label: 1,
          value: 1,
        },
        {
          label: 2,
          value: 2,
        },
        {
          label: 3,
          value: 3,
        },
        {
          label: 4,
          value: 4,
        },
        {
          label: 5,
          value: 5,
        },
        {
          label: 6,
          value: 6,
        },
      ],
    };
  },
};
```

```topic
multiple
```

```desc
multiple `value` is `String` or `Number`
multiple `options` 2d array, item of the secondary array is `Object`, key should be `label`、`value`
```

```html
<byted-picker title="title" v-model="value" :options="options" />
```

```javascript
  data() {
    return {
      value: [1, 1],
      options: [

          { label: 3, value: 3 },

          { label: 0, value: 0 },

          { label: 2, value: 2 },
          { label: 2.5, value: 2.5 },
          { label: 3, value: 3 },
        ],
      ],
    };
  },
};
```

```topic
With Popup
```

```html
<byted-cell title="Area" value="Choose" is-link :border="false" @click="show=!show" />
<byted-popup v-model="show" position="bottom">
  <byted-picker
    title="Area"
    v-model="value"
    :options="options"
    @cancel="show=!show"
    @confirm="confirm"
  />
</byted-popup>
```

```javascript
export default {
  methods: {
    confirm() {
      this.show = !this.show;
      this.$toast(`the value : ${JSON.stringify(this.value)}`);
    },
  },
};
```

## API

### Props

| Attribute           | Description             | Type                        | Default   |
| ------------------- | ----------------------- | --------------------------- | --------- |
| value               | Key of option value     | _string \| number \| array_ | `''`      |
| options             | array                   | _array_                     | `[]`      |
| title               | title                   | _string_                    | -         |
| show-toolbar        | Whether to show toolbar | _boolean_                   | `true`    |
| confirm-button-text | Text of confirm button  | _string_                    | `Cancel`  |
| cancel-button-text  | Text of cancel button   | _string_                    | `Confirm` |
| show-cancel-button  | show cancel button      | _boolean_                   | `true`    |

## Events

### Picker Events

| Event   | Description                           | Arguments |
| ------- | ------------------------------------- | --------- |
| change  | Triggered when current option changed | `value`   |
| input   | Triggered when current option changed | `value`   |
| confirm | Triggered when click confirm button   | Event     |
| cancel  | Triggered when click cancel button    | Event     |
