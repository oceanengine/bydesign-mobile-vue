# Stepper

### install

:::card

```javascript
import { Stepper } from '@byted/by-design-m';

Vue.use(Stepper);
```

:::

## Usage

:::card

```topic
basic usage
```

```html
<byted-stepper v-model="value1" size="small" :min="1" :max="100" :step="10" />
<byted-stepper v-model="value2" size="normal" />
<byted-stepper v-model="value3" size="large" />
<byted-stepper v-model="value4" size="large" :disabled="true" />
<script>
  export default {
    data() {
      return {
        value1: 1,
        value2: 3,
        value3: 4,
        value4: 5,
      };
    },
    methods: {
      changeValue(value) {
        console.log(value);
      },
    },
  };
</script>
```

:::

## API

### Props

| Attribute | Description                            | Type      | Default |
| --------- | -------------------------------------- | --------- | ------- |
| size      | Can be set to `large` `normal` `small` | `string`  | `md`    |
| v-model   | stepper's value                        | `number`  | `1`     |
| disabled  | stepper can be use                     | `boolean` | `false` |
| min       | stepper's min value                    | `number`  | `0`     |
| max       | stepper's max value                    | `num`     | `null`  |
| step      | stepper's step value                   | `num`     | `1`     |

## Events

| Event  | Description                  | Arguments            |
| ------ | ---------------------------- | -------------------- |
| change | Triggered after value change | value: current value |
