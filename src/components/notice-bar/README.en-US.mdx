# NoticeBar

### Install

:::card

```javascript
import { NoticeBar } from '@byted/by-design-m';

Vue.use(NoticeBar);
```

:::

## Usage

:::card

```topic
Basic Usage
```

```html
<byted-notice-bar text="title is what is title" />
<byted-notice-bar text="title is what is titletitle is what is title" />
```

:::

:::card

```topic
Close Usage
```

```html
<byted-notice-bar show-close text="title is what is title" />
<byted-notice-bar show-close text="title is what is titletitle is what is title" />
```

:::

:::card

```topic
Ellipsis Usage
```

```html
<byted-notice-bar :ellipsis="1" show-close text="title is what is titletitle is what is title" />
<byted-notice-bar
  :ellipsis="2"
  text="title is what is titletitle is what is titletitle is what is titletitle is what is title"
/>
```

:::

:::card

```topic
Style Usage
```

```html
<byted-notice-bar show-close type="primary"> title is what is title </byted-notice-bar>
<byted-notice-bar show-close type="success"> title is what is title </byted-notice-bar>

```

:::

## API

### Props

| Attribute  | Description                                                | Type      | Default   |
| ---------- | ---------------------------------------------------------- | --------- | --------- |
| text       | Notice text content                                        | `string`  | `''`      |
| type       | Type, can be set to `primary` `success` `warning` `danger` | `string`  | `warning` |
| show-close | Show close icon                                            | `boolean` | `false`   |
| ellipsis   | Open ellipsis                                              | `number`  | `0`       |

### Events

| Event | Description                    | Arguments    |
| ----- | ------------------------------ | ------------ |
| click | Triggered when click NoticeBar | event: Event |
| close | Triggered when closed          | event: Event |

### Slots

| Name    | Description         |
| ------- | ------------------- |
| default | Notice text content |
| left    | Custom left         |
| right   | Custom right        |
