# Mobile BUI

`Mobile BUI`是基于 <strong>ByDesign</strong> 设计体系的移动端 Vue UI 组件库。目前 ByDesign 提供了 Vue 2 版本、React 版本、Taro 小程序版本。

---

### 特性

- 提供 <strong>35</strong> 个高质量组件，覆盖移动端各类场景
- 组件单元测试覆盖率达 <strong>90%</strong>，为稳定性提供保障
- 支持国际化，兼容 <strong>Vue-i18n</strong>，支持 <strong>16</strong> 种语言
- 支持自定义品牌名称
- 支持TypeScript
- 支持主题定制
- 支持按需引入

### 开源协议

`Mobile BUI`基于 <strong>MIT</strong> 协议，可自由参与开源。
